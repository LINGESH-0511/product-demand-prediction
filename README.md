# 📊 Product Demand Prediction System

> **Predict future product demand using 3 Machine Learning models - Linear Regression, Random Forest, and XGBoost**

## 🎯 Problem Statement

Shop owners struggle with two main problems:

| Problem | Consequence |
|---------|-------------|
| **Overstocking** | Money wasted, products expire, storage costs increase |
| **Understocking** | Lost sales, angry customers, damaged reputation |

**Solution:** A machine learning system that predicts exactly how many units will be sold in the future.

## ✨ Features

- 📁 **Upload CSV** - Upload your sales data easily
- 🤖 **3 ML Models** - Linear Regression, Random Forest, XGBoost
- 🏆 **Auto Model Selection** - System picks the best performing model
- 📊 **Data Statistics** - See total records, unique stores, top products
- 🔮 **Demand Prediction** - Predict sales for 7, 14, 21, 28, 30, 45, 60, 90 days
- 📥 **Download CSV** - Save predictions for record keeping
- 📱 **Responsive UI** - Works on desktop, tablet, and mobile
